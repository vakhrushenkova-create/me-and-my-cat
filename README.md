# Мой любимый кот Кузьма
Мой котик очень красивый и ласковый! 
    
    Его любимые занятия: 
1. Играть со мной в догонялки
2. Катать свои мятные шарики по дому
3. Гулять на окне и рассматривать собак 
4. Ужинать вместе со мной 

Он очень любит когда я даю ему лакомства и становится после них еще ласковее. 
Обычно он спит со мной и мило сварачивается в клубочек
